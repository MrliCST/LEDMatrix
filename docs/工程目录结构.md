# EasyMatrix 工程目录结构

```
EasyMatrix/
├── platformio.ini            # PlatformIO 项目配置（芯片、框架、依赖库）
├── diagram.json              # Wokwi 仿真接线图（引脚来源）
├── data/
│   └── index.html            # 配网页面 HTML 模板（上传到 SPIFFS）
├── docs/
│   ├── 工程目录结构.md         # ← 本文件
│   └── 页面开发指南.md         # 面向页面开发者的入门文档
│
├── include/                  # 头文件（接口声明 + 数据）
│   ├── core/                 # 核心配置 & 基础设施
│   ├── hal/                  # 硬件抽象层
│   ├── pages/                # 页面模块
│   ├── net/                  # 网络模块
│   ├── img/                  # 位图 & 动画帧数据
│   └── songs/                # 乐谱数据
│
└── src/                      # 实现文件（接口实现）
    ├── main.cpp              # 入口：setup() + loop()
    ├── core/                 # 核心模块实现
    ├── hal/                  # 硬件驱动实现
    ├── pages/                # 页面逻辑实现
    └── net/                  # 网络模块实现
```

---

## 一、根目录文件

| 文件 | 说明 |
|---|---|
| `platformio.ini` | 项目配置：芯片 `esp32dev`，框架 `arduino`，依赖库列表 |
| `diagram.json` | Wokwi 仿真接线定义，**所有引脚以此文件为准** |

---

## 二、include/core/ — 核心配置 & 基础设施

> 全局共用，所有模块都依赖这里。

| 文件 | 说明 |
|---|---|
| `config.h` | **引脚定义**、屏幕尺寸、亮度范围、动画帧率、NTP 服务器、枚举（`Page`、`TimeMode`、`RhythmMode` 等） |
| `store.h` | **全局数据仓库**（单例）：WiFi 密码、颜色、子模式、闹钟、亮度。对外只读，写入必须走 `saveXxx()` 方法，自动持久化到 NVS |
| `state_machine.h` | **页面状态机**（单例）：管理 6 个页面，负责页面切换（`gotoPage`）、每帧派发（`update`）、按键事件派发（`onButton`） |

---

## 三、include/hal/ — 硬件抽象层 (Hardware Abstraction Layer)

> 把硬件操作封装成简单的方法调用。**页面开发者只调这里，不直接碰硬件**。

| 文件 | 对应硬件 | 主要方法 |
|---|---|---|
| `led_matrix.h` | 4× 8×8 NeoPixel 矩阵（32×8 屏） | `clear()` `drawPixel()` `print()` `show()` `color()` |
| `microphone.h` | 模拟麦克风（ADC） | `sample()` → `bands[32]` FFT 频段数据 |
| `buzzer.h` | 无源蜂鸣器（PWM） | `playSong(index)` `stop()` `isPlaying()` |
| `buttons.h` | 3 个物理按键 | `begin()` `tick()` — 自动检测长短按，推给 StateMachine |

---

## 四、include/pages/ — 页面模块

> 每个页面是一个类，继承 `PageBase`。  
> **组员主要工作在这里。** 头文件已写好注释，`.cpp` 里填实现。

| 文件 | 页面 | 说明 |
|---|---|---|
| `page_base.h` | 基类 | 定义 `enter()` `exit()` `update()` + 6 个按键虚函数 |
| `setting_page.h` | 配网 | WiFi AP + WebServer 配网页 |
| `time_page.h` | 时间显示 | NTP 时间，3 种格式切换 |
| `rhythm_page.h` | 节奏灯 | 麦克风 FFT 频谱，4 种可视化 |
| `anim_page.h` | 动画 | 3 种装饰动画（雨滴/渐变/闪烁） |
| `clock_page.h` | 闹钟 | 设置闹钟时间/铃声/开关 |
| `bright_page.h` | 亮度 | 亮度步进调节 (5~145) |

**页面跳转链路**：
```
SETTING ─→ TIME ─→ RHYTHM ─→ ANIM ─→ CLOCK ─→ BRIGHT ─→ TIME
```

---

## 五、include/net/ — 网络模块

> 配网页面和 NTP 对时的底层网络支持。

| 文件 | 说明 |
|---|---|
| `wifi_manager.h` | STA 连接 + AP 热点 + 连接状态 |
| `ntp.h` | NTP 对时 + 定时器 + 闹钟剩余秒数计算 |
| `web_server.h` | HTTP 服务（配网页路由 `/` + `/configwifi`） |

---

## 六、include/img/ — 位图 & 动画帧数据

> 只暴露 `img.h`，原始数据文件在 `elems/` 子目录。

```
img/
├── img.h                  ← 唯一入口，include 这个即可
└── elems/
    ├── animHack.h             数字雨 6 色调色板 (uint16_t)
    ├── bell.h                 闹钟图标 (6×8 单色)
    ├── checkingTime.h         对时中提示 (32×8 单色)
    └── timeAnim0 ~ 8.h        时间翻页动画 9 帧 (uint16_t RGB)
```

`img.h` 提供的 API：

| 符号 | 类型 | 说明 |
|---|---|---|
| `animHack[]` | `const uint16_t*` | 色板数据 |
| `IMG_BELL` | `MonoBitmap` | 闹钟图标 {data, 6, 8} |
| `IMG_CHECKING_TIME` | `MonoBitmap` | 对时提示 {data, 32, 8} |
| `TIME_ANIM_FRAMES[9]` | `const uint16_t*[]` | 翻页动画 9 帧数组 |

---

## 七、include/songs/ — 乐谱数据

> 只暴露 `songs.h`，原始数据文件在 `elems/` 子目录。

```
songs/
├── songs.h                 ← 唯一入口，include 这个即可
└── elems/
    ├── notes.h                 音符频率宏 (NOTE_B0 ~ NOTE_DS8)
    ├── super_mario.h           超级马里奥
    ├── always_with_me.h        Always With Me
    ├── dream_wedding.h         梦中的婚礼
    ├── castle_in_the_sky.h     天空之城
    └── canon.h                 卡农
```

`songs.h` 提供的 API：

| 符号 | 类型 | 说明 |
|---|---|---|
| `SONG_LIST[5]` | `SongInfo[]` | 歌曲查找表 {notes, durations, length} |
| `SONG_COUNT` | `#define` | 歌曲总数 = 5 |

---

## 八、src/ — 实现文件

> 每个 `include/xxx/yyy.h` 对应一个 `src/xxx/yyy.cpp`。  
> **页面开发者只需要改 `src/pages/` 下的对应 `.cpp`。**

| 目录 | 说明 | 谁负责 |
|---|---|---|
| `src/main.cpp` | `setup()` 初始化所有模块，`loop()` 驱动按键+状态机 | **核心组** |
| `src/core/` | `store.cpp`（NVS 读写）、`state_machine.cpp`（页面切换逻辑） | **核心组** |
| `src/hal/` | 硬件驱动实现（矩阵、麦克风、蜂鸣器、按键） | **核心组** |
| `src/pages/` | **6 个页面的实现** — 组员填这里 | **页面组** |
| `src/net/` | WiFi、NTP、WebServer 实现 | **核心组** |

---

## 九、角色的工作范围

### 核心组

| 职责 | 涉及文件 |
|---|---|
| 引脚/常量定义 | `include/core/config.h` |
| 数据仓库 | `include/core/store.h` + `src/core/store.cpp` |
| 状态机 | `include/core/state_machine.h` + `src/core/state_machine.cpp` |
| 硬件驱动 | `include/hal/*.h` + `src/hal/*.cpp` |
| 网络 | `include/net/*.h` + `src/net/*.cpp` |
| 入口 | `src/main.cpp` |
| 配网页 | `data/index.html` |

### 页面组

| 职责 | 涉及文件 |
|---|---|
| 页面基类 | `include/pages/page_base.h`（只读） |
| 6 个页面头文件 | `include/pages/*_page.h`（只读，已写好注释） |
| 6 个页面实现 | `src/pages/*_page.cpp`（**在这里写代码**） |

页面组只需知道的头文件：

```cpp
#include "pages/page_base.h"       // 基类
#include "core/config.h"            // 常量/枚举（通过 page_base.h 间接引入）
#include "core/store.h"             // 读写配置
#include "core/state_machine.h"     // 页面跳转
#include "hal/led_matrix.h"         // 画屏
#include "img/img.h"                // 位图
#include "songs/songs.h"            // 乐谱
```

## 十、数据流全景图

```
┌──────────────────────────────────────────────────────────────────┐
│  loop() 每 10ms                                                   │
│    │                                                              │
│    ├──→ Buttons.tick() ──→ StateMachine.onButton()              │
│    │                              │                               │
│    │                              └──→ 当前 Page::onBtnXxx()      │
│    │                                     │                        │
│    │                                     ├── 读/写 Store          │
│    │                                     ├── 调 LedMatrix 画屏    │
│    │                                     ├── 调 Buzzer 播放      │
│    │                                     └── gotoPage() 跳转     │
│    │                                                              │
│    └──→ StateMachine.update()                                     │
│             │                                                     │
│             └──→ 当前 Page::update()                              │
│                    │                                              │
│                    ├── 读 Store::mode() / brightness()            │
│                    ├── 读 Microphone::bands()（仅 RhythmPage）    │
│                    └── 调 LedMatrix 画屏                          │
└──────────────────────────────────────────────────────────────────┘
```
